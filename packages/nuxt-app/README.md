# Nuxt Application

Main data management platform application built with Nuxt 3 and consuming the Linarc Design System.

## Development

```bash
pnpm dev
```

## Build

```bash
pnpm build
```

## Features

- Nuxt 3 with TypeScript
- Pinia for state management
- VueUse composables
- Tailwind CSS styling
- Design System integration
- Auto-imports for components and composables
